## 📖 DESKRIPSI

Tool spam NGL paling **sultan** dan **anti-ketahuan** yang pernah ada! Dibuat dengan ❤️ pake Python, didesain buat lo yang pengen nge-prank temen, mantan, atau siapapun yang NGL-nya sepi kayak kuburan. 

**Fitur unggulan:**
- 🔄 Auto retry kalo gagal (pantang menyerah!)
- 🎭 Rotate User-Agent biar gak ketahuan bot
- 🎲 Random Device ID tiap request
- ⏱️ Custom delay biar makin stealth
- 📊 Real-time progress tracker
- 💻 Cross-platform (Windows, Linux, Termux, MacOS)
- 🛡️ Anti-debug & encrypted payload
